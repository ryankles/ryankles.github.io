AVC Interactive Map App

Tools:

Led a team of 10+ student developers in creating an interactive mobile application designed to enhance navigation for AVC students and staff. The project was initiated with $10,000 in funding from AVC President/Superintendent Dr. Zellot and began as an Android application built using Android Studio and Java. The Google Maps SDK was successfully integrated, utilizing its FusedLocationProvider API along with Android's Location API to display users' real-time locations and dynamically reveal points of interest (POIs) based on user proximity. A user-friendly interface was designed, incorporating key features such as GPS integration and class schedule reminders. To ensure cross-platform compatibility, the application was migrated to Flutter for iOS deployment, with a scheduled launch on both the App Store and Google Play in Spring 2025.
